# 김아영의 유니버스 🌏🪐⭐

HTML, CSS, JavaScript를 활용해 제작한 개인 웹페이지입니다.
프로필, 학습 내용, 여행 기록과 다양한 JavaScript 기능을 제공합니다.

## 주요 기능

* 👤 개인 프로필 소개
* 📅 수업 시간표
* 🌴 휴일 계획
* 📸 여행 앨범 (사진 · 오디오 · 동영상)
* 📝 회원가입 페이지
* 🌤️ Open-Meteo API를 활용한 실시간 날씨 검색 및 조회
* 🚀 업다운 게임
* 📊 성적 계산기
* 🎒 내 가방 보기
* 🔮 랜덤 오늘의 운세

## 기술 스택

* HTML5
* CSS3
* JavaScript (ES6)
* Flexbox / Grid
* Fetch API
* Open-Meteo API

## 프로젝트 구조

```text
skala-front/

├── css/
├── script/
│   ├── upDown.js
│   ├── grade.js
│   ├── bag.js
│   ├── fortune.js

│   └── weatherAPI.js
└── media/
```

## 실행 방법

```bash
git clone https://github.com/kay25802/skala-front.git
cd skala-front
```

VS Code에서 프로젝트를 연 뒤 **Live Server**를 실행하고 `html/index.html`을 열어 확인합니다.
